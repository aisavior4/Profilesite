## 🎯 상황별 가이드

### 📌 **처음 사이트를 시작할 때**

```
1단계: README.md 읽기
→ 프로젝트 이해하기

2단계: DEPLOYMENT-GUIDE.md 읽기
→ 사이트 배포하기

3단계: USER-MANUAL.md 읽기
→ 관리 방법 익히기

4단계: QUICK-REFERENCE.md 북마크
→ 빠른 참조용
```

---

### 📌 **콘텐츠를 수정하고 싶을 때**

```
1단계: USER-MANUAL.md 해당 섹션 찾기
→ "콘텐츠 수정하기" 또는
→ "이미지 관리하기" 또는
→ "이력서 관리하기"

2단계: 가이드대로 수정

3단계: Publish 탭 → Deploy

4단계: 확인 (Hard Refresh)
```

---

### 📌 **빠르게 작업하고 싶을 때**

```
QUICK-REFERENCE.md 열기
→ 원하는 작업 찾기
→ 단계별로 따라하기
→ Deploy → 확인
```

---

### 📌 **문제가 생겼을 때**

```
1단계: USER-MANUAL.md의 "문제 해결" 섹션 확인
→ FAQ에서 같은 문제 찾기

2단계: QUICK-REFERENCE.md의 "문제 해결" 확인
→ 빠른 해결 방법

3단계: 그래도 안 되면
→ 제니에게 질문! 😊
```

---

### 📌 **도메인을 구매하고 싶을 때**

```
DEPLOYMENT-GUIDE.md 열기
→ "4단계: 도메인 구매 및 연결" 섹션
→ 가이드대로 따라하기
```

---

### 📌 **Google에 등록하고 싶을 때**
