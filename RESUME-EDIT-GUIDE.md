# 📝 Resume 기존 데이터 수정 완벽 가이드

## 🎯 목적
현재 Resume 페이지(resume.html)에 표시되는 **기존 이력 데이터를 수정**하는 방법을 안내합니다.

---

## ✅ 수정 가능한 항목

Resume 페이지의 **모든 섹션**을 admin-simple.html에서 수정할 수 있습니다:
- 학력
- 국제공인 자격증 (13개)
- 사회경력
- 저서
- 논문
- 주요 정부사업 및 프로젝트
- 음향과정 교육 워크샵
- 그 외 특강
- 전문가 심의 및 평가활동
- NCS 개발 및 심의·교육경력

---

## 📖 단계별 수정 방법

### **1단계: Admin 페이지 열기**
```
파일 열기: admin-simple.html
```

### **2단계: "이력서" 탭 클릭**
- 페이지 상단의 **"이력서"** 버튼 클릭
- 현재 저장된 모든 이력 항목이 **행(Row) 형태**로 표시됩니다

### **3단계: 카테고리 필터 사용** (선택사항)

페이지 상단의 **"카테고리"** 드롭다운 메뉴에서 선택:

| 선택 항목 | 표시되는 내용 |
|----------|-------------|
| **전체 보기** | 모든 이력 항목 |
| **학력** | 학력 항목만 |
| **국제공인 자격증** | 자격증 항목만 |
| **사회경력** | 경력 항목만 |
| **저서** | 저서 항목만 |
| **논문** | 논문 항목만 |
| **주요 정부사업 및 프로젝트** | 프로젝트 항목만 |
| **음향과정 교육 워크샵** | 워크샵 항목만 |
| **그 외 특강** | 특강 항목만 |
| **전문가 심의 및 평가활동** | 심의 항목만 |
| **NCS 개발 및 심의·교육경력** | NCS 관련 항목만 |

**예시**: "학력"만 수정하고 싶다면 → **"학력"** 선택 → 학력 항목만 표시됨

### **4단계: 수정할 항목 찾기**

이력 목록은 다음과 같은 **행(Row) 형태**로 표시됩니다:

```
┌─────────────────────────────────────────────────────────┐
│ [순서 1]  제목: 세종대학교 일반대학원                    │
│          기관: 음악학과 실용음악전공                     │
│          날짜: 2024                                      │
│          설명: 음악학 박사                               │
│          [⭐ Featured]  [수정] [삭제]                   │
└─────────────────────────────────────────────────────────┘
```

- **순서 번호**: 왼쪽에 큰 숫자로 표시 (Resume 페이지 표시 순서)
- **⭐ Featured**: 강조 표시 항목
- **[수정]**: 클릭하면 수정 가능
- **[삭제]**: 클릭하면 삭제 (복구 불가)

### **5단계: "수정" 버튼 클릭**

수정할 항목의 **"수정"** 버튼을 클릭하면:
1. 페이지 최상단으로 자동 스크롤
2. 입력 폼에 **기존 데이터가 자동으로 로드**됨
3. **"✏️ 수정 모드"** 메시지 표시

### **6단계: 내용 수정**

입력 폼에서 다음 항목을 수정할 수 있습니다:

| 필드 | 설명 | 예시 |
|------|------|------|
| **카테고리** | 이력 분류 | 학력, 자격증, 경력 등 |
| **제목** | 주요 제목 | "세종대학교 일반대학원" |
| **부제목** | 소제목 (기관명 등) | "음악학과 실용음악전공" |
| **날짜** | 연도 또는 기간 | "2024", "2020-2022" |
| **설명** | 상세 설명 | "음악학 박사 학위 취득" |
| **순서** | 표시 순서 (숫자) | 1, 2, 3... (작을수록 먼저 표시) |
| **강조 표시** | Featured 체크박스 | 중요 항목 체크 |

### **7단계: "저장" 버튼 클릭**

- 수정 후 **"저장"** 버튼 클릭
- 성공 메시지: **"✅ 이력서 항목이 저장되었습니다!"**
- 페이지가 자동으로 새로고침되어 수정 내용 반영

### **8단계: Resume 페이지 확인**

1. `resume.html` 파일 열기
2. **Ctrl+F5** (강제 새로고침)
3. 수정한 내용이 정상적으로 표시되는지 확인

## 🎯 실전 예시

### **예시 1: 학력 항목 수정**

**목표**: 세종대학교 학위 연도를 2023 → 2024로 변경

1. admin-simple.html 열기
2. "이력서" 탭 클릭
3. 카테고리에서 **"학력"** 선택
4. "세종대학교 일반대학원" 항목 찾기
5. **"수정"** 버튼 클릭
6. **"날짜"** 필드를 "2023" → **"2024"** 로 변경
7. **"저장"** 클릭
8. resume.html에서 확인

### **예시 2: 자격증 항목 삭제**

**목표**: 더 이상 유효하지 않은 자격증 삭제

1. admin-simple.html 열기
2. "이력서" 탭 클릭
3. 카테고리에서 **"국제공인 자격증"** 선택
4. 삭제할 자격증 찾기
5. **"삭제"** 버튼 클릭
6. 확인 메시지: **"정말 삭제하시겠습니까?"**
7. **"확인"** 클릭
8. resume.html에서 확인

### **예시 3: 프로젝트 순서 변경**

**목표**: 최신 프로젝트를 맨 위로 이동

1. admin-simple.html 열기
2. "이력서" 탭 클릭
3. 카테고리에서 **"주요 정부사업 및 프로젝트"** 선택
4. 최신 프로젝트 찾기
5. **"수정"** 버튼 클릭
6. **"순서"** 필드를 "10" → **"1"** 로 변경 (작을수록 먼저 표시)
7. 다른 프로젝트들의 순서도 필요 시 조정 (2, 3, 4...)
8. **"저장"** 클릭
9. resume.html에서 순서 확인

### **예시 4: Featured (강조) 설정**

**목표**: 중요한 경력을 강조 표시

1. admin-simple.html 열기
2. "이력서" 탭 클릭
3. 카테고리에서 **"사회경력"** 선택
4. 강조할 경력 찾기
5. **"수정"** 버튼 클릭
6. **"강조 표시"** 체크박스 체크 ✅
7. **"저장"** 클릭
8. resume.html에서 확인 (강조된 스타일로 표시됨)

## 💡 유용한 팁

### 🔍 **순서(Order) 필드 활용법**
- **작은 숫자 = 먼저 표시**
- **큰 숫자 = 나중에 표시**
- 예: Order 1 → Order 2 → Order 3 순서로 표시

**권장 방법**:
- 10단위로 순서 지정 (10, 20, 30...)
- 나중에 중간에 항목 추가 시 편리 (예: 15, 25 추가 가능)

### ⭐ **Featured (강조) 사용법**
- 중요한 항목에 체크
- Resume 페이지에서 시각적으로 강조됨
- 너무 많이 체크하면 효과 감소 (3-5개 권장)

### 📋 **카테고리 필터 활용**
- 많은 항목 중 특정 분류만 보고 싶을 때 사용
- 예: "학력"만 수정 → "학력" 필터 선택 → 빠르게 찾기

### 🔄 **데이터 백업**
- 중요한 수정 전에는 현재 내용을 메모장에 복사해두기
- 삭제된 데이터는 **복구 불가능**

## ⚠️ 주의사항

### ❌ **절대 하지 말아야 할 것**
1. **"삭제" 버튼을 실수로 클릭하지 마세요**
   - 삭제된 데이터는 복구 불가능
   - 삭제 전 반드시 확인

2. **같은 순서 번호 사용하지 마세요**
   - Order 1이 여러 개면 표시 순서가 꼬임
   - 각 항목마다 고유한 순서 번호 사용

3. **필수 필드를 비워두지 마세요**
   - 카테고리, 제목은 필수
   - 비워두면 저장 안 됨

### ✅ **권장 사항**
1. 수정 전 Ctrl+F5로 페이지 새로고침
2. 수정 후 resume.html에서 바로 확인
3. 카테고리 필터를 적극 활용
4. 순서는 10단위로 지정 (확장성)

## 🚨 문제 해결

### Q1: 수정했는데 resume.html에 반영이 안 돼요
**A**: Ctrl+F5로 강제 새로고침하세요. 브라우저 캐시 때문일 수 있습니다.

### Q2: "수정" 버튼을 눌러도 아무 반응이 없어요
**A**: 
1. F12 콘솔에서 에러 메시지 확인
2. 페이지를 새로고침하고 다시 시도
3. 다른 브라우저에서 시도

### Q3: 카테고리 필터가 작동하지 않아요
**A**: 페이지를 새로고침(F5)하고 다시 선택해보세요.

### Q4: 순서를 바꿨는데 resume.html에서 순서가 안 바뀌어요
**A**: 
1. "저장" 버튼을 눌렀는지 확인
2. resume.html에서 Ctrl+F5 강제 새로고침
3. admin에서 순서 번호가 제대로 저장되었는지 확인

### Q5: 새 항목을 추가하고 싶어요
**A**: 
1. 페이지 상단의 입력 폼에서 직접 작성
2. 카테고리, 제목 등 입력
3. "저장" 클릭
4. 하단 목록에 추가된 것 확인
