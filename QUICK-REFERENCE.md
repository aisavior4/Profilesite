# 🚀 빠른 참조 가이드 (Quick Reference)

**자주 하는 작업들을 한눈에!**

---

## 📌 관리자 페이지 접속

```
https://your-site.netlify.app/admin-simple.html
```

---

## ⚡ 자주 하는 작업

### 1️⃣ 이미지 업로드
```
갤러리 탭 → 파일 선택 → 카테고리 선택 → 
제목 입력 → Featured 체크 → 갤러리 추가 → 
Publish 탭 → Deploy
```

### 2️⃣ 프로필 사진 변경
```
프로필 관리 탭 → Hero/About 이미지 업로드 → 
파일 선택 → 업로드 → Publish 탭 → Deploy
```

### 3️⃣ 학력 수정
```
About Me 학력 탭 → 학력 1~5 입력 → 
[대괄호] 사용 → 학력 저장 → 
Publish 탭 → Deploy
```

### 4️⃣ 기술 스택 수정
```
Resume 헤더 탭 → Left/Right Skills 입력 → 
쉼표로 구분 → Resume 헤더 저장 → 
Publish 탭 → Deploy
```

### 5️⃣ Expertise 추가/수정
```
Expertise 관리 탭 → 제목/아이콘/설명/태그 입력 → 
순서 선택 → Featured 체크 → 
Expertise 추가 → Publish 탭 → Deploy
```

### 6️⃣ 이력서 추가
```
이력서 탭 → 카테고리 선택 → 
제목/부제목/날짜/설명 입력 → 
순서 선택 → Featured 체크 → 
이력서 추가 → Publish 탭 → Deploy
```

### 7️⃣ 게시글 작성
```
게시판 탭 → 제목/본문 입력 → 
이미지 업로드(선택) → 카테고리 선택 → 
게시글 추가 → Publish 탭 → Deploy
```

### 8️⃣ 순서 정렬
```
해당 섹션의 "순서 자동 정렬" 버튼 클릭 → 
확인 → Publish 탭 → Deploy
```

---

## 🔄 배포 (가장 중요!)

```
✅ 수정 후 반드시 해야 함!

Publish 탭 → Deploy 버튼 클릭 → 
1-2분 대기 → 사이트 새로고침 (Ctrl+Shift+R)
```

---

## 🐛 문제 해결

### 변경사항이 반영 안 됨
```
1. Publish 탭 → Deploy 클릭
2. Hard Refresh (Ctrl+Shift+R)
3. 시크릿 모드로 확인
```

### 순서가 엉망
```
"순서 자동 정렬" 버튼 클릭 → Deploy
```

### 이미지 업로드 실패
```
1. 이미지 크기 5MB 이하로 압축
2. TinyPNG.com에서 압축
3. 다시 업로드
```

### 데이터 안 보임
```
1. admin-simple.html 열기
2. "현재 데이터 불러오기" 클릭
3. 데이터 확인
4. Deploy → Hard Refresh
```

---

## ⚠️ 주의사항

```
❌ 절대 하지 말 것:
- admin-simple.html 코드 직접 수정
- 여러 브라우저에서 동시 수정
- Deploy 없이 수정만 하고 확인

✅ 항상 할 것:
- 수정 후 반드시 Deploy
- 순서 꼬이면 "자동 정렬" 버튼 클릭
- 중요 내용은 별도 백업
- Hard Refresh로 확인
```

```

2️⃣ 저장 버튼 클릭
3️⃣ Publish 탭 → Deploy
4️⃣ 사이트 Hard Refresh (Ctrl+Shift+R)
```
