```

□ sitemap.xml

□ js 폴더 (전체)
□ images 폴더 (전체)
```

---

## 🚀 2단계: Netlify 배포 (3분)

```
□ https://www.netlify.com 접속
□ "Sign up" 클릭
□ GitHub 계정으로 가입
□ 이메일 인증 완료
□ https://app.netlify.com/drop 접속
□ 프로젝트 폴더 드래그 앤 드롭
□ 업로드 완료 대기
□ URL 받기!
```

---

## 🎯 3단계: URL 변경 (1분)

```
□ Netlify 대시보드 → 사이트 클릭
□ "Site settings" 클릭
□ "Change site name" 클릭
□ 원하는 이름 입력 (예: limhyungjun)
□ "Save" 클릭
□ 새 URL 확인
```

---

## ✅ 4단계: 확인 (5분)

```
□ 브라우저에서 URL 접속
□ 메인 페이지 확인
□ About Me 섹션 확인
□ Expertise 카드 확인
□ Gallery 확인
□ Resume 확인
□ Publications 확인
□ 관리자 페이지 확인 (URL/admin-simple.html)
□ 모바일에서 확인 (F12 → Ctrl+Shift+M)
□ 친구들에게 공유! 🎉
```
