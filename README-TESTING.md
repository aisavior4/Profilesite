# 테스트 가이드

## ⚠️ 중요: 반드시 이 순서대로 테스트하세요

### 1. Admin 페이지 접속
```
admin-simple.html
```

### 2. 갤러리 이미지 업로드 테스트

#### 단계:
1. "클릭하여 이미지 선택" 영역 클릭
2. 이미지 파일 1개 선택
3. 미리보기에 이미지 표시 확인
4. 폼 작성:
   - 제목: "테스트 이미지"
   - 카테고리: "Studio" 선택
   - 설명: "테스트"
   - 대표 이미지: 체크
5. **"저장" 버튼 클릭**
6. **초록색 성공 메시지 확인**: "1개 이미지가 저장되었습니다!"
7. 하단 "기존 이미지"에 방금 추가한 이미지 확인

#### 예상 결과:
- ✅ 성공 메시지 표시
- ✅ 하단에 "테스트 이미지" 카드 표시
- ✅ "Studio" 카테고리 표시
- ✅ "⭐ 대표 이미지" 표시

#### 실패 시:
- ❌ 빨간색 에러 메시지 표시
- 브라우저 F12 → Console 탭에서 에러 확인

### 3. index.html 확인

1. `index.html` 열기
2. Gallery 섹션으로 스크롤
3. **Studio 카드 확인**:
   - 대표 이미지가 표시되어야 함
   - "1 photos" 카운트 표시

### 4. 게시판 테스트

1. `admin-simple.html` → "게시판" 탭 클릭
2. 폼 작성:
   - 제목: "테스트 게시글"
   - 내용: "테스트 내용입니다."
   - 작성자: "관리자"
   - 상단 고정: 체크
3. **"게시글 저장" 클릭**
4. 성공 메시지 확인
5. 하단에 게시글 표시 확인

### 5. board.html 확인

1. `board.html` 열기
2. 방금 작성한 게시글 표시 확인
3. 게시글 클릭
4. 상세 페이지 표시 확인

---

## 🐛 문제 발생 시

### 증상 1: "저장" 클릭해도 반응 없음
- F12 → Console 탭 확인
- 빨간색 에러 메시지 확인
- 에러 내용 복사해서 알려주세요

### 증상 2: "기존 이미지"에 표시 안 됨
- 페이지 새로고침 (F5)
- 여전히 안 보이면 에러 보고

### 증상 3: index.html에 이미지 안 보임
- admin-simple.html에서 "대표 이미지" 체크했는지 확인
- 카테고리가 정확한지 확인 (studio, work, education, publication)
