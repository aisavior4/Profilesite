## 🎯 기본 사용법

### 📌 **관리자 페이지 접속**

```
1. 브라우저에서 사이트 열기
2. 주소창에 /admin-simple.html 추가
   예: https://your-site.netlify.app/admin-simple.html
3. 관리자 페이지 열림!
```

### 📌 **관리자 페이지 구조**

관리자 페이지에는 7개 탭이 있어요:

```
📁 갤러리 - 이미지 업로드/수정/삭제
📁 프로필 관리 - Hero 이미지, About Me 프로필 수정
📁 Expertise 관리 - 전문 분야 6개 카드 관리
📁 About Me 학력 - About Me 섹션 학력 5개 관리
📁 Resume 헤더 - 이름, 직함, 기술 스택 관리
📁 Publications & Media - 논문, 기사, 기타 관리
📁 이력서 - 학력, 자격증, 경력, 프로젝트 등 관리
📁 게시판 - 공지사항, 일반 게시글 관리
```

## ✏️ 콘텐츠 수정하기

### 1️⃣ **프로필 이미지 및 소개글 수정**

#### **Hero 섹션 (메인 페이지 최상단) 수정:**

```
1. admin-simple.html 열기
2. "프로필 관리" 탭 클릭
3. Hero Image 섹션:
   - "파일 선택" → 원하는 이미지 선택
   - "Hero 이미지 업로드" 클릭
4. 완료! 메인 페이지 새로고침하면 변경됨
```

#### **About Me 프로필 수정:**

```
1. "프로필 관리" 탭
2. About Me Profile 섹션:
   - "파일 선택" → 프로필 사진 선택
   - "About 이미지 업로드" 클릭
3. 이름 입력: "임형준"
4. 소개글 입력:
   "음악학 박사이자 Dolby ATMOS & Pro Tools 마스터 트레이너..."
5. "프로필 저장" 클릭
6. 완료!
```

---

### 2️⃣ **About Me 학력 수정**

```
1. "About Me 학력" 탭 클릭
2. 학력 1~5 입력:
   예: [음악학 박사] 세종대학교 일반대학원 음악학과 실용음악전공
3. "학력 저장" 클릭
4. 완료!

⚠️ 주의: 반드시 [대괄호]로 학위를 감싸주세요!
예: [음악학 박사], [석사], [Diploma]
```

---

### 3️⃣ **기술 스택 (Tech Stack) 수정**

```
1. "Resume 헤더" 탭 클릭
2. Left Skills (왼쪽 열):
   예: Dolby ATMOS, Recording, Pro Tools, Mixing
   (쉼표로 구분)
3. Right Skills (오른쪽 열):
   예: DANTE, Mastering, Cubase, Producing
4. "Resume 헤더 저장" 클릭
5. 완료!

💡 팁: 이름과 직함은 비워도 됨 (기술만 수정 가능)
```

---

### 4️⃣ **Expertise (전문 분야) 수정**

#### **새 Expertise 추가:**

```
1. "Expertise 관리" 탭 클릭
2. 제목 입력: 예) "Pro Tools"
3. 아이콘 선택:
   - Font Awesome 아이콘 사용
   - 예: fas fa-sliders-h
   - 아이콘 찾기: https://fontawesome.com/icons
4. 설명 입력:
   "Pro Tools 국제공인 ACI 및 저서 집필..."
5. 스킬 태그 입력:
   Recording, Mixing, Editing (쉼표로 구분)
6. 순서: 드롭다운에서 선택 (1-10)
7. "메인 페이지에 표시" 체크
8. "Expertise 추가" 클릭
9. 완료!
```

#### **기존 Expertise 수정:**

```
1. 기존 Expertise 목록에서 "수정" 버튼 클릭
2. 내용 수정
3. "수정 저장" 클릭
4. 완료!
```

#### **Expertise 삭제:**

```
1. "삭제" 버튼 클릭
2. 확인 대화상자에서 "확인" 클릭
3. 완료!
```

#### **순서 자동 정렬:**

```
1. "순서 자동 정렬" 버튼 클릭 (우측 상단)
2. 자동으로 1, 2, 3... 순서로 정렬됨
3. 완료!

💡 팁: 순서가 꼬이면 이 버튼을 누르세요!
```

## 🖼️ 이미지 관리하기

### 1️⃣ **갤러리 이미지 업로드**

```
1. "갤러리" 탭 클릭
2. 카테고리 선택:
   - Sound Production (레코딩 믹싱 프로듀싱)
   - Education (교육 & 세미나)
   - Work & People (일과 사람들)
   - Life (좋아하는 것)
3. "파일 선택" → 이미지 여러 개 선택 가능
4. 제목 입력: 예) "스튜디오 녹음 세션"
5. 설명 입력 (선택사항)
6. 순서: 드롭다운에서 선택 (1-20)
7. "Featured (대표 이미지)" 체크 (메인 페이지에 표시)
8. "갤러리 추가" 클릭
9. 업로드 중... (진행률 표시)
10. 완료!
```

**⚠️ 이미지 제한:**
- 파일 크기: 최대 5MB (자동 압축됨)
- 권장 해상도: 1200px 이하
- 지원 형식: JPG, PNG, GIF

---

### 2️⃣ **갤러리 이미지 수정**

```
1. 기존 이미지 목록에서 "수정" 버튼 클릭
2. 제목, 설명, 순서 수정
3. 이미지 교체 (선택사항):
   - "파일 선택" → 새 이미지 선택
4. "수정 저장" 클릭
5. 완료!

💡 팁: 이미지를 바꾸지 않으려면 "파일 선택"을 누르지 마세요!
```

---

### 3️⃣ **갤러리 이미지 삭제**

```
1. "삭제" 버튼 클릭
2. 확인 대화상자에서 "확인" 클릭
3. 완료!
```

---

### 4️⃣ **갤러리 순서 정렬**

```
방법 1: 드롭다운에서 순서 선택
1. "수정" 버튼 클릭
2. 순서 드롭다운에서 원하는 번호 선택
3. "수정 저장" 클릭

방법 2: 자동 정렬 (추천!)
1. "순서 자동 정렬" 버튼 클릭 (우측 상단)
2. 현재 표시된 순서대로 1, 2, 3... 자동 정렬
3. 완료!

💡 팁: 순서가 꼬이거나 중복되면 "자동 정렬" 버튼을 누르세요!
```

## 🔄 사이트 업데이트 배포하기

### 📌 **수정 후 사이트 업데이트 방법**

콘텐츠를 수정했으면 반드시 **재배포**해야 변경사항이 실제 사이트에 반영됩니다!

#### **방법 1: Publish 탭 사용 (가장 쉬움!) ⭐⭐⭐⭐⭐**

```
1. 화면 위쪽 "Publish" 탭 클릭
2. "Deploy" 또는 "재배포" 버튼 클릭
3. 1-2분 대기
4. 완료! 변경사항이 실제 사이트에 반영됨
5. 사이트 접속해서 확인 (Ctrl+Shift+R로 새로고침)
```

**💡 이게 제일 쉬워요! 항상 이 방법을 사용하세요!**

---

#### **방법 2: Netlify 수동 재배포**

Netlify를 직접 사용 중이라면:

```
1. Netlify 로그인
2. 사이트 선택
3. "Deploys" 탭 클릭
4. "Trigger deploy" → "Deploy site" 클릭
5. 또는 수정된 파일을 드래그 앤 드롭으로 재업로드
6. 완료!
```

---

#### **방법 3: GitHub 자동 배포 (설정된 경우)**

GitHub과 연동했다면:

```
1. GitHub 저장소에 파일 업로드/수정
2. Commit & Push
3. Netlify가 자동으로 재배포 시작
4. 1-2분 후 완료!
```

---

### 📌 **배포 후 확인하기**

```
1. 사이트 접속
2. Hard Refresh (캐시 삭제 새로고침):
   - Windows: Ctrl + Shift + R
   - Mac: Cmd + Shift + R
3. 변경사항 확인:
   ✅ 새 이미지가 표시되는지
   ✅ 수정한 텍스트가 반영되었는지
   ✅ 순서가 올바른지
4. 모바일에서도 확인 (F12 → Ctrl+Shift+M)
```

## 🔧 문제 해결

### ❓ **자주 묻는 질문 (FAQ)**

#### **Q1: 이미지를 업로드했는데 사이트에 안 나와요**

```
A: 체크리스트:
□ "Featured (대표 이미지)" 체크했나요?
□ 올바른 카테고리를 선택했나요?
□ 업로드 후 "재배포" 했나요?
□ 브라우저 캐시 삭제 (Ctrl+Shift+R)했나요?
□ F12 Console에서 오류 확인
```

---

#### **Q2: 순서가 엉망이에요**

```
A: "순서 자동 정렬" 버튼을 누르세요!
1. 해당 섹션(Expertise, Gallery, Publications, Resume)의
   "순서 자동 정렬" 버튼 클릭
2. 자동으로 1, 2, 3... 정렬됨
3. 완료!
```

---

#### **Q3: 수정했는데 사이트에 반영이 안 돼요**

```
A: 재배포를 잊으셨나요?
1. Publish 탭 → Deploy 버튼 클릭
2. 1-2분 대기
3. 사이트 Hard Refresh (Ctrl+Shift+R)
4. 확인!

💡 중요: 관리자 페이지에서 수정한 내용은 
   데이터베이스에 저장되지만, 
   실제 사이트에 반영되려면 "재배포"가 필요합니다!
```

---

#### **Q4: 이미지가 너무 커서 업로드가 안 돼요**

```
A: 이미지 압축 방법:
1. 온라인 압축 사이트 사용:
   - TinyPNG: https://tinypng.com
   - Compressor.io: https://compressor.io
2. 이미지 크기 조절:
   - 권장: 1200px 이하
   - 최대: 1920px
3. 파일 크기: 5MB 이하로 압축
4. 다시 업로드
```

---

#### **Q5: 학력이 About Me에 표시 안 돼요**

```
A: 체크리스트:
□ "About Me 학력" 탭에서 입력했나요?
  (이력서 탭이 아닙니다!)
□ 대괄호를 사용했나요?
  예: [음악학 박사] 세종대학교...
□ 재배포 했나요?
□ Hard Refresh (Ctrl+Shift+R) 했나요?
```

---

#### **Q6: 기술 스택이 About Me에 안 나와요**

```
A: 체크리스트:
□ "Resume 헤더" 탭에서 입력했나요?
□ 쉼표(,)로 구분했나요?
  예: Dolby ATMOS, Pro Tools, Cubase
□ "Resume 헤더 저장" 버튼 클릭했나요?
□ 재배포 했나요?
□ Hard Refresh 했나요?
```

---

#### **Q7: Console에 오류가 떠요**

```
A: F12를 눌러 Console 확인:
1. 404 오류: 파일이 없음
   → 해당 파일을 확인하고 재배포
2. API 오류: 데이터 로딩 실패
   → 관리자 페이지에서 데이터 확인
3. 기타 오류:
   → 오류 메시지 캡처해서 제니에게 문의
```

---

#### **Q8: 모바일에서 제목이 잘려요**

```
A: 이미 해결되었어요!
□ 최신 버전으로 재배포했나요?
□ 스마트폰 브라우저 캐시 삭제:
  Safari → 설정 → 방문 기록 및 데이터 지우기
□ Private 브라우징 모드로 확인
```

### 🆘 **긴급 문제 해결**

#### **사이트가 완전히 안 열려요!**

```
1. URL 확인:
   □ https:// 포함했나요?
   □ 철자가 맞나요?
2. Netlify 상태 확인:
   □ Netlify.com 로그인
   □ Deploys 탭에서 상태 확인
   □ 실패했다면 "Retry deploy" 클릭
3. 브라우저 확인:
   □ 다른 브라우저에서 열어보기
   □ 시크릿 모드로 열어보기
```

---

#### **데이터가 전부 사라졌어요!**

```
⚠️ 데이터는 데이터베이스에 저장되므로 절대 사라지지 않습니다!

확인 방법:
1. admin-simple.html 열기
2. 각 탭에서 데이터 확인
3. "현재 데이터 불러오기" 버튼 클릭
4. 데이터가 보이면 정상입니다
5. 메인 페이지에서 안 보인다면:
   → Publish 탭에서 재배포
   → Hard Refresh (Ctrl+Shift+R)
```
