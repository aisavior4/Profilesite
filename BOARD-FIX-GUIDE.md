# 📋 Board 게시글 본문이 안 보이는 문제 해결 가이드

## ⚠️ 문제 상황
- Board 페이지에서 게시글 목록은 보임
- 게시글을 클릭하면 **제목만 보이고 본문이 비어있음** (흰 바탕만 표시)

---

## 🔍 원인
게시글을 작성할 때 **"내용" 필드가 비어있거나 공백만 입력된 경우** 발생합니다.

---

## ✅ 해결 방법 (단계별 가이드)

### **1단계: Admin 페이지 열기**
```
파일 열기: admin-simple.html
```

### **2단계: 게시판 탭 클릭**
- 페이지 상단의 **"게시판"** 버튼 클릭
- 기존 게시글 목록이 표시됩니다

### **3단계: 본문이 비어있는 게시글 확인**
게시글 목록에서 다음과 같은 표시를 찾으세요:
```
⚠️ 내용: ⚠️ 본문 없음 - 수정 필요!
```

이런 게시글이 **Board 페이지에서 본문이 안 보이는 게시글**입니다.

### **4단계: 게시글 수정**
1. 해당 게시글의 **"수정"** 버튼 클릭
2. 상단 입력 폼에 기존 데이터가 로드됩니다
3. **"내용"** 필드를 찾으세요 (아래 이미지 참고)
4. 본문 내용을 작성합니다

**예시**:
```
안녕하세요, 임형준입니다.

이번에 새로운 소식을 전해드립니다.
자세한 내용은 아래를 참고해주세요.

감사합니다.
```

### **5단계: 저장**
- **"게시글 저장"** 또는 **"저장"** 버튼 클릭
- 성공 메시지 확인

### **6단계: Board 페이지 확인**
1. `board.html` 페이지로 이동
2. **Ctrl+F5** (강제 새로고침)
3. 수정한 게시글 클릭
4. 본문이 정상적으로 표시되는지 확인

---

## 🎯 새 게시글 작성 시 주의사항

### ✅ **올바른 작성 방법**
```
1. 제목 입력: "공지사항"
2. 내용 입력: "본문 내용을 반드시 입력하세요!" ⭐ 중요!
3. 작성자 입력: "관리자"
4. 이미지 첨부 (선택사항)
5. 상단 고정 체크 (선택사항)
6. "저장" 클릭
```

### ❌ **잘못된 작성 방법**
```
1. 제목만 입력
2. 내용 필드 비워둠 ❌
3. "저장" 클릭
→ Board에서 본문이 안 보임!
```

## 💡 디버깅 방법

### **F12 콘솔 확인**
1. Board 페이지에서 **F12** 누르기
2. **Console** 탭 클릭
3. 게시글 클릭
4. 콘솔 로그 확인:

```javascript
🔍 Full Post Object: {
  "id": "...",
  "title": "공지사항",
  "content": "",  // ⚠️ 비어있음!
  "author": "관리자",
  ...
}
📝 Post.content value: undefined
📄 Post.content length: 0
```

만약 `content`가 비어있으면 → **admin에서 수정 필요!**

## 🚨 자주 묻는 질문 (FAQ)

### Q1: 게시글을 저장했는데 본문이 안 보여요
**A**: "내용" 필드를 비워두고 저장했기 때문입니다. admin-simple.html에서 해당 게시글을 수정하고 본문을 추가하세요.

### Q2: 모든 게시글의 본문이 비어있어요
**A**: 게시글을 작성할 때 "내용" 필드를 입력하지 않았기 때문입니다. 각 게시글을 하나씩 수정해서 본문을 추가하세요.

### Q3: 본문을 입력했는데도 안 보여요
**A**: 
1. 본문에 공백만 입력하지 않았는지 확인
2. Board 페이지에서 Ctrl+F5 강제 새로고침
3. F12 콘솔에서 `post.content` 값 확인

### Q4: HTML 태그를 사용할 수 있나요?
**A**: 네! 본문에 HTML 태그를 사용할 수 있습니다.

**예시**:
```html
<h2>제목</h2>
<p>본문 내용</p>
<ul>
  <li>항목 1</li>
  <li>항목 2</li>
</ul>
```
